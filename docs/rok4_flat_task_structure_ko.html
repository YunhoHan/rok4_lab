Action pipelineкіј clip

RoK4 policyк°Җ м¶ңл Ҙн•ҳлҠ” actionмқҖ лЁјм Җ RSL-RL wrapperм—җм„ң clip_actions = 1.0 м„Өм •мқ„ нҶөн•ҙ [-1, 1] лЎң мһҳлҰ°лӢӨ. к·ё лӢӨмқҢ RoK4 action scaleкіј default pose offsetмқҙ м Ғмҡ©лҗҳм–ҙ мӢӨм ң joint position targetмқҙ лҗңлӢӨ.

actor network output
  -> RslRlVecEnvWrapper clip_actions = 1.0
  -> clipped_raw_action in [-1, 1]
  -> q_target = default_joint_pos + clipped_raw_action * ROK4_ACTION_SCALE
  -> IdealPDActuatorCfg explicit torque-PD

мӨ‘мҡ”н•ң м җмқҖ last_action observationмқҙ q_target мқҙлӮҳ clipped_raw_action * ROK4_ACTION_SCALE мқҙ м•„лӢҲлқј clipped raw policy actionмқҙлқјлҠ” м җмқҙлӢӨ. мҰү observation м•Ҳмқҳ action н•ӯлӘ©мқҖ "мқҙм „ stepм—җм„ң policyк°Җ лӮё action"мқ„ мқҳлҜён•ҳкі , motor targetмқҖ лі„лҸ„лЎң action scaleкіј default poseлҘј кұ°міҗ л§Ңл“Өм–ҙм§„лӢӨ.

л°ҳл©ҙ action_rate_l2 мҷҖ second_action_rate_l2 rewardлҠ” raw action м°ЁмқҙлҘј к·ёлҢҖлЎң м“°м§Җ м•Ҡкі , clipped_raw_action * ROK4_ACTION_SCALE м°ЁмқҙлҘј мӮ¬мҡ©н•ңлӢӨ. мҰү observationмқҖ raw action мқҳлҜёлҘј мң м§Җн•ҳкі , smoothness rewardлҠ” мӢӨм ң joint target offset ліҖнҷ”лҹүм—җ лҚ” к°Җк№Ңмҡҙ к°’мңјлЎң кі„мӮ°н•ңлӢӨ.

ROK4_ACTION_SCALE мқҖ кё°мЎҙ Isaac Gym RoK4мқҳ actuator action scaleкіј лҸҷмқјн•ҳлӢӨ. мўҢмёЎ лӢӨлҰ¬, мҡ°мёЎ лӢӨлҰ¬, torso мҲңм„ңмқҳ к°’мқҖ лӢӨмқҢкіј к°ҷлӢӨ.

н•ҷмҠө command лІ”мң„
